# Edesia
## Get all the recipes!!!

### Setup
1. Install IDE such as VSCode or PyCharm
2. Install requirements (pip install -r requirements.txt)
3. Run python edesia.py

### Current Websites
1. allrecipes.com
2. food.com
3. cookbook.com
4. seriouseats.com

Using this to scrape: https://github.com/hhursev/recipe-scrapers
